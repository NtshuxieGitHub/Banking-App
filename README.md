# Banking-App
Banking application with management dashboard using Next.js

# Tech Stack
- Next.js
- TypeScript
- Appwrite
- Plaid
- Dwolla
- React Hook Form
- Zod
- TailwindCSS
- Chart.js
- ShadCN

# Application Features
1. Authentication
2. Connect Banks
3. Home Page
4. My Banks
5. Transaction History
6. Real-Time Updates
7. Funds Transfer
8. Responsiveness

# Pre-requisites
Ensure that the following is installed on your machine:

- Git
- Node.js
- NPM (Node Package Manager)
